**Playwright/Cypress - Explicación Rápida**

Son herramientas de testing automatizado para web:

| Tool | Lenguaje | Ventaja |
|------|----------|---------|
| Playwright | JS/Python | Más rápido, multi-browser |
| Cypress | JS | Más fácil, mejor debug visual |

**Cómo funcionan:**
```javascript
// Ejemplo Playwright
await page.goto('https://anfitrion-mx.vercel.app');
await page.fill('#tarifaNoche', '2000');
await page.fill('#numNoches', '5');
await page.click('#btnCalcular');
expect(await page.textContent('#gananciaNeta')).toBe('$6,100');
```

**Diferencia con Chrome MCP:**
- MCP: Claude piensa → ejecuta → piensa (lento, ~5 seg/acción)
- Playwright: Script corre directo (~100ms/acción)

**Mi propuesta:**
1. Claude + MCP = Diseñar pruebas (inteligente)
2. Playwright = Ejecutar pruebas (rápido)
3. CI/CD = Correr en cada deploy (automático)
