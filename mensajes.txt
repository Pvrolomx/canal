**Ejemplo concreto Playwright:**

Cuando corre `npx playwright test`:

```
Running 2 tests using 1 worker

  ✓ suma 5 + 3 = 8 (1.2s)
  ✓ suma 100 + 200 = 300 (0.8s)

2 passed (2.5s)
```

**Qué hace internamente (en 2 segundos):**
1. Abre Chromium headless (invisible)
2. Navega a la URL
3. Llena campo #num1 con '5'
4. Llena campo #num2 con '3'
5. Click en #btnSumar
6. Lee resultado
7. Compara: ¿es '8'? → ✓ PASS

**El problema en RPi:** Chromium ARM tiene issues de red. Funciona mejor en PC/Mac.

**Alternativa rápida:** GitHub Actions corre Playwright gratis en cada push.
