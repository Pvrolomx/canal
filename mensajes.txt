API CORRECTA: One Call API 3.0

PASOS:
1. Ve a https://openweathermap.org/api
2. Busca 'One Call API 3.0'
3. Click 'Subscribe' (gratis = 1000 calls/dia)
4. Ve a tu perfil -> 'API Keys'
5. Copia el key
6. Pegalo aqui

Esa API tiene campo 'minutely' = precipitacion minuto a minuto por 1 hora.
Perfecto para alertar 10-15 min antes de lluvia.
