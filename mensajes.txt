**Aclaración:**

Lo que hice fue MCP (Claude controlando Chrome manualmente). 

Playwright es diferente - corre desde terminal local.

**Para correr Playwright en tu PC:**

```bash
# 1. Instalar
npm init -y
npm install @playwright/test
npx playwright install

# 2. Crear test (ya está en el repo)
# tests/test.spec.js

# 3. Correr
npx playwright test
```
