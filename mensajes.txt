=== SPEC: POMODORO TIMER ===

NOMBRE: FocusFlow
COMPLEJIDAD: Media (mas que ColorSnap, menos que QuickSplit)
TIEMPO ESTIMADO: 15-20 min

DESCRIPCION:
Timer Pomodoro minimalista con sonido y estadisticas de sesion.

FUNCIONALIDADES:

1. TIMER PRINCIPAL
   - 25 min trabajo (configurable)
   - 5 min descanso corto
   - 15 min descanso largo (cada 4 pomodoros)
   - Botones: Start, Pause, Reset, Skip

2. VISUAL
   - Circulo animado que se llena/vacia
   - Colores: Rojo trabajo, Verde descanso
   - Numero grande con minutos:segundos
   - Indicador de pomodoro actual (1/4, 2/4, etc)

3. SONIDO
   - Notificacion cuando termina cada fase
   - Usar Web Audio API o audio HTML5

4. ESTADISTICAS (en sesion)
   - Pomodoros completados hoy
   - Tiempo total de enfoque
   - Se guarda en localStorage

5. CONFIGURACION (opcional)
   - Ajustar duracion de trabajo/descanso
   - Toggle sonido on/off

STACK:
- Next.js 14 (App Router)
- Tailwind CSS
- Sin backend, todo frontend
- Deploy a Vercel

CRITERIO DE EXITO:
- URL funcionando en Vercel
- Timer cuenta regresiva correctamente
- Sonido al terminar
- Estadisticas persisten en localStorage
